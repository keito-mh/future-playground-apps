# カラーミッション

## 概要

四方から流れてくる色玉を、指で囲って消していくスマホ向けアクションゲームです。

毎レベルで「消すべき色」が1つ指定され、その色だけを狙って消します。指定数を消すとレベルアップし、画面上の玉がいっせいに弾けて体力が全回復、さらに数秒間どの色でも消し放題の「ボーナスタイム」に突入します。

## 目的

スコア・失敗・連続成功・難易度上昇を1つの操作（囲う／タップ）に集約し、「1回遊んだら、すぐもう1回」を生むこと。消したときの弾ける気持ちよさと、レベルアップ時の全消し＋ボーナスの爽快感を中心に据えています。

## 流れ

1. `index.html` を開く
2. 上部中央に表示される「ターゲット色」を確認する
3. その色の玉をタップ、または指でぐるっと囲って消す
4. スコア、残りミッション数、ライフ、コンボが変化する
5. ミッションを達成するとレベルアップ → 全玉が弾けて体力全回復 → ボーナスタイム
6. ボーナスタイム中はどの色でも消し放題でスコアを稼ぐ
7. ライフが尽きるとゲームオーバー
8. 結果（到達レベル・スコア）を見て、もう一度挑戦する

## 基本ルール

- ライフ: 3（ハート表示）
- ターゲット色を消す: スコア加点、コンボ継続、ミッション進行
- 違う色を消す: ライフ−1、コンボリセット
- 連続成功: コンボ倍率で加点アップ
- レベルアップ: 全玉消去 + 体力全回復 + ボーナスタイム（約5秒・消し放題・スコア2倍）
- 難易度: レベルが上がるほど玉が増え・速くなり、演出（ブルーム→花火→ネオン→レインボー）が派手になる

## 操作

- スマホ: 玉をタップ、または指でなぞって囲む
- PC: 玉をクリック、またはドラッグして囲む

## ビジュアル方向

- 暗い宇宙空間に発光する色玉が漂う、ネオン／パーティクル基調
- 消去時は加算合成のバースト、衝撃波リング、流れ星で「弾ける」快感を演出
- レベルが上がるほど背景の星雲グローと粒子が増え、画面が派手に進化

## リプレイ設計

スコア／ライフ（ゲージ管理）／失敗状態／コンボ／難易度上昇／ボーナス（リスクリワードと報酬）／60秒未満の短セッションを内包し、AGENTS.md のリプレイ要件を満たします。

## スクリーンショット／シェアポイント

- レベルアップ時の全玉いっせい爆発 + 「LEVEL UP」バナー
- ボーナスタイムの虹色 + 大量バースト
- ゲームオーバー画面の「到達LEVEL・SCORE」

## 安全設計

純粋な色当てアクションで、診断・性格判定・健康情報などは扱いません。個人情報は取得せず、ログイン・外部API・データ保存・バックエンドはありません。

## 実装ノート

- 素のHTML / CSS / JavaScript（Canvas 2D）、依存ライブラリなし
- モバイルファースト、全画面・セーフエリア対応、ピンチズーム無効化
- 単一の `<canvas>` に全描画。状態は1つの IIFE 内で管理

## 起動方法

ブラウザで `index.html` を開いてください。
