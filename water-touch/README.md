# みずなぞり

指で画面をなぞると、水面が軌跡に沿ってスッとひらく、触って気持ちいい水面体験。
ゲームではなく「触る・見る・何度もなぞる」ための体験型ミニアプリです。

## あそびかた

- **なぞる / スワイプ** — 水面が軌跡に沿ってひらき、波紋が広がります。
- **タップ** — 小さな波紋がふわっと生まれます。
- **速さで変化**
  - ゆっくり：ぬるっと水が分かれる
  - 速い：スパッと白い筋が走り、水しぶきが散る
- 水面は **1〜2秒で自然に静まる**ので、何度でもなぞれます。

## モード

| モード | 質感 |
| --- | --- |
| クリア | 透明感のある基本の水 |
| フロー | 一定方向にゆっくり流れる水 |
| ジェリー | ぷるっと戻る重めの水 |
| きらめき | 光の粒がたくさん散る水 |

光の粒や葉っぱが波にあわせてゆっくり動きます（触感を邪魔しない控えめな演出）。

- 右上の雫アイコン … 水面をしずめる（リセット）
- 画面下部 … モード切替

## しくみ（軽量な疑似シミュレーション）

物理的に正確な流体計算はしていません。軽く動くことを最優先に、

- 低解像度の**高さ場（height field）** で波の伝播と自然な戻りを表現
- グリッドを `ImageData` で描き、滑らかに拡大して水面に
- 波紋リング・水しぶき・白い筋・光の粒・葉っぱは Canvas 2D のオーバーレイ

で構成しています。依存ライブラリなし、`index.html` をブラウザで開くだけで動きます。

## ファイル

- `index.html` — マークアップ
- `style.css` — レイアウトとUI
- `script.js` — 水面シミュレーションと演出
