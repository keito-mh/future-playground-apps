# みずなぞり

最初は、真っ暗で静かな夜の水辺。指でなぞって光をともし、集めて咲かせるほど、
**水辺が少しずつ育っていきます** — 水面が明るくなり、星が増え、蓮が咲き、光の魚が泳ぎだす。
触って気持ちいい水面体験に、「自分の手で世界を生き返らせる」手応えを重ねた体験型ミニアプリです。

## 育つ水辺（この体験の芯）

- **暗い水を、光で満たす** — はじまりは光のほとんどない夜。なぞって光を集め、咲かせるほど「いのち」が貯まります。
- **育つほど世界が変わる** — 水面が明るくなり、夜空に星がふえ、水辺に蓮が咲き、やがて光の魚が泳ぎだします。育ちは積み重なって戻りません。
- **節目の合図** — 一定数咲かせるごとに、短い言葉（「星がふえた」「蓮がさいた」…）とともに世界が一段育ちます。
- リセット（右上の雫）で、また夜の水辺から育て直せます。

## あそびかた

- **なぞる / スワイプ** — 水面が軌跡に沿ってひらき、波紋が広がります。
- **タップ** — 小さな波紋がふわっと生まれます。
- **速さで変化**
  - ゆっくり：ぬるっと水が分かれる
  - 速い：スパッと白い筋が走り、水しぶきが散る
- 水面は **1〜2秒で自然に静まる**ので、何度でもなぞれます。
- **タッチで色がにじむ** — 触れた場所からやわらかい色がふわっと広がり、ゆっくり水色へ戻ります。

## 光をあつめる（そっと添えたゲーム性）

「指を動かすこと自体が気持ちいい」に拍車をかける、失敗のない遊びです。

- **なぞると、曲が流れる** — 指で触れている間、その曲が**正しいリズムで一定に流れます**。タイミングを当てる必要はなく、なぞるだけで「ちゃんと演奏している」感覚に。タップ＝その場で1音。
- **音が光で見える** — 1音ごとに指先で小さな光がポンと咲き、波紋が広がります。
- **光が指についてくる／集めると育つ** — 漂う光の粒が指へ集まり、咲かせるほど水辺が育ちます（暗→明・星・蓮・魚）。
- **曲はモードで変わる（→「モード」参照）。切り替えると新しい曲の出だしが自動で流れて、変わったのがすぐ分かります。**
- **ときどき特別演出** — 何回か咲かせると、ランダムな間隔・ランダムな種類で特別な演出が現れます（大輪の花／光のシャワー／虹の波紋／ホタルの舞）。

音はすべてその場で合成（音源ファイル不要・すべてパブリックドメイン曲）。スコアもタイマーも失敗もありません。左上のスピーカーで音はオフにできます。

## モード（見た目だけでなく、遊んだ感触が変わります）

| モード | 曲 | 体験のちがい |
| --- | --- | --- |
| クリア | きらきら星 | 透明感のある基本の水。澄んでキビキビ反応。 |
| フロー | かえるのうた | 水に流れがある。光も葉もぐんぐん流れ、止まると指先から流される。 |
| ジェリー | メリーさんのひつじ | 重くてぷるぷる。波がゆっくり大きく揺れ戻り、光の寄りも粘って遅い。 |
| きらめき | よろこびの歌 | 光がたくさん・速く湧き、なぞると光の粒子が散る。特別演出も出やすい。 |

各モードで、**流れる曲（パブリックドメイン）**・波の重さ・光の集まり方・湧く速さ・特別演出の頻度・音色まで変わります。モードを切り替えると曲も変わり、その曲のリズムで咲かせると演奏になります。

- 左上のスピーカー … 音のオン/オフ
- 右上の雫アイコン … 水面をしずめる（リセット）
- 画面下部 … モード切替

## しくみ（軽量な疑似シミュレーション）

物理的に正確な流体計算はしていません。軽く動くことを最優先に、

- 低解像度の**高さ場（height field）** で波の伝播と自然な戻りを表現
- グリッドを `ImageData` で描き、滑らかに拡大して水面に
- 波紋リング・水しぶき・白い筋・光の粒・葉っぱは Canvas 2D のオーバーレイ

で構成しています。依存ライブラリなし、`index.html` をブラウザで開くだけで動きます。

## ファイル

- `index.html` — マークアップ
- `style.css` — レイアウトとUI
- `script.js` — 水面シミュレーションと演出
