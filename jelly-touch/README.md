# ぷるぷるゼリー

画面中央の大きな半透明ゼリーを、タップ・ドラッグ・長押し・スマホシェイクでぷるぷる揺らして遊ぶ、触って気持ちいいミニ体験。スコアも勝敗もありません。触っているうちに色や中の泡・光の粒・揺れ方がじわっと育ち、最後に「自分だけのゼリー」が完成します。

## 遊び方

- **タップ** — タップした所がぷにっとへこんで、揺れが広がる。泡や光がすこし出て、色がほんの少し変わる。
- **ドラッグ / 引っ張る** — 指に合わせてゼリーがぐにーっと伸びる。伸びすぎる前に弾力で抵抗し、離すとぷるんと戻る。
- **長押し** — ゼリーがむにっと沈み、離すと大きくぷるんと跳ねる。透明感が増していく。
- **シェイク** — 端末を軽く揺らすと全体がぷるぷる震える。揺らし続けると震えが持続し、やめても余韻でしばらく揺れる。加速度センサーが使えない時は画面下の「ぷるぷる」ボタンで代用。
- **♪おんがく** — 画面下の「♪おんがく」をオンにすると、やさしいループに合わせてゼリーが拍ごとにぷるんと脈動し、ふちが光る（初期はオフ）。

触り方の傾向によって、ゼリーの**色（二色グラデの配色）・輪郭の形・中の泡や光の粒（連続タップで星形に）・揺れ方**が少しずつ変わり、見ただけで見分けがつく「自分だけのゼリー」に育ちます。完成するゼリーのタイプ（ぷるぷる／もちもち／きらきら／にじいろ／とうめい）と名前も変わります。下部に「完成する」ボタンが出たら、押すと完成演出と名前が表示されます。

ゼリーは触れている時だけ反応します（外側から押してもくっついてきません）。素早く引いて離すほど大きく弾み、長く押し込むほど強く跳ね返ります。

## 操作のヒント

- 上部のメーターは「ぷるぷるメーター」。触るほど伸びて、いっぱいになると「完成する」ボタンが出ます。
- 左上は音のオン・オフ（初期はミュート）。右上は最初からやり直すボタン。
- 完成画面では「またぷるぷるする」でそのまま遊び続けられ、「もう一度つくる」で新しいゼリーを作れます。

## 技術メモ

- 素の HTML / CSS / JavaScript。Canvas 2D のみ。ビルド不要。
- 揺れは「固有モード（mode2=楕円, mode3, mode4, mode5 の定在波）」を減衰調和振動子で表現。低いモードほどゆっくり・高いモードほど速く減衰するので、本物のゼリーのように「ぷるるん…」と上品に収まる。
- タップ＝接点を押し込み（局所くぼみ＋まわりがふくらむ）→離すとモードを励起してぷるん、ドラッグ＝軸方向に体積保存で伸びるアフィン変形、長押し＝スカッシュ、シェイク＝モードへ微小エネルギーを足し続けて持続＋余韻。
- シェイクは `DeviceMotion` を低めのしきい値で検知（iOS は最初のボタン操作で許可要求）。非対応環境は「ぷるぷる」ボタンで代替。
- スマホ縦画面前提。`touch-action: none` でスクロール暴発を防止。子どもにも安全な明るい配色。
