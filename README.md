# Future Playground

遊べるミニ体験の公開コレクション。各アプリは素のHTML / CSS / JavaScriptで動作し、GitHub Pagesで公開しています。

## 公開アプリ

- [カラーミッション](./color-mission/) — 流れてくる色玉を指で囲って消すアクションゲーム。お題の色だけを狙い、レベルアップで囲い放題のボーナスタイムに突入。
- [みずなぞり](./water-touch/) — 指でなぞると水面がスッとひらく、触って気持ちいい水面体験。波紋・水しぶき・光の粒が広がり、質感は4モードで切替。

## 公開URL

GitHub Pages: `https://keito-mh.github.io/<このリポジトリ名>/`

## 構成

- `index.html` — アプリ一覧のハブページ
- `<app-name>/` — 各アプリ（`index.html` / `style.css` / `script.js` / `README.md`）

新しいアプリを追加するときは、フォルダを1つ作って `index.html` を置き、ハブの `index.html` にカードを1枚足すだけです。
